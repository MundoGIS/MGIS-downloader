LMV – Geotorget, Developer Portal & STAC

Följ dessa steg för att få teknisk åtkomst till Lantmäteriets STAC-tjänster.

Viktigt: Använd ett token genererat i API Manager eller ett systemkonto skapat i Geotorget.
Exempel (kontroll av collections med curl): 
# Exempel med systemkonto (user + api key):
curl -u "SYSTEMUSER:API_KEY" -H "X-API-Key: API_KEY" "https://api.lantmateriet.se/stac-vektor/v1/collections"

# Partial GET med systemkonto:
curl -u "SYSTEMUSER:API_KEY" -H "X-API-Key: API_KEY" -H "Range: bytes=0-1023" "https://api.lantmateriet.se/path/to/asset.tif"

# Exempel med Bearer token (Auth token):
curl -H "Authorization: Bearer " "https://api.lantmateriet.se/stac-vektor/v1/collections"

# Partial GET med Bearer token:
curl -H "Authorization: Bearer " -H "Range: bytes=0-1023" "https://api.lantmateriet.se/path/to/asset.tif"

Byt ut SYSTEMUSER och API_KEY mot ditt systemkontonamn och den nyckel/secret som du fått från din organisation eller via Geotorget.

  1. 1. Skapa konto på Geotorget
    Besök Geotorget och registrera din organisation. Detta krävs för att teckna avtal för de datamängder ni vill använda (t.ex. Fastighetskartan eller Höjddata).
  2. 2. Skapa en App i Developer Portal
    Logga in på Developer Portal med dina Geotorget-uppgifter.
    • Gå till "Applications" → "Add New Application".
    • Sök upp API:erna (t.ex. STAC Vektor) och klicka på "Subscribe".
  3. 3. Generera ett Access Token med rätt scopes
    I din Application i Developer Portal:
    1. Välj Production Keys (eller rätt miljö för din access).
    2. Klicka Select Scopes och markera scopes som ger åtkomst till STAC-resurser — t.ex. scopes för collections och asset-read, eller ett bredare scope som ger read‑access.
    3. Klicka på Generate Access Token eller motsvarande knapp för att skapa tokenet.
    4. Kopiera tokenet (access_token) och testa det med curl eller i applikationen (välj "Auth token" och klistra in token).

    Om din organisation istället använder systemkonto, hämta systemkonto‑användarnamn och tillhörande API‑nyckel från Geotorget och använd "Användarnamn + API‑nyckel" i appen.

  4. 4. API-slutpunkter & STAC Browser
    Använd länkarna nedan för att ansluta din applikation. Du kan även öppna dem direkt i Radian Earth STAC Browser för att inspektera datat visuellt:

    STAC Vektor (t.ex. Fastighetskartan):

    https://api.lantmateriet.se/stac-vektor/v1 Öppna Browser

    STAC Höjd (t.ex. Markmodeller, Laserdata):

    https://api.lantmateriet.se/stac-hojd/v1 Öppna Browser

Tips: När du öppnar STAC Browser via länkarna ovan kommer den att be om inloggning. Du kan prova med antingen Consumer Key/Secret eller med ett Bearer token.
För produktionstester rekommenderas att du genererar ett token med rätt scopes i Developer Portal och använder det i applikationen (välj "Auth token").

GBIF – Artdata

Instruktioner för att hämta artobservationer via GBIF.

  1. Skapa konto
    Registrera dig på GBIF.org.
  2. Användning
    Ange dina GBIF-inloggningsuppgifter i appen för att ladda ner data. Kom ihåg att citera källan med det DOI-nummer som genereras vid varje uttag.

Vid tekniska problem med nycklar, besök Lantmäteriets supportguider.
